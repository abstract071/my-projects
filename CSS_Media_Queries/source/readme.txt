1. Browser that you should support: latest version of Chrome, Firefox, Safari, IE10+

2. Create a responsive layout using relative length units (%, em) based on design in .psd file
To convert fixed unit to relative please use the formula (target ÷ context) * 100% = result  

3. Your web site should be responsive to all screen resolution on desktop version and for mobile devices such as iPad and iPhone
You should support minimal screen resolution is 320*480 and everything then bigger.

4. You can use to check your work a Chrome plugin 'Resize Window'.
Also to check for iPad or iPhone you can enable emulation in the web inspector and choose there needed device.
